## target-ecommerce-sql
Target E-commerce Data Analysis Case Study By SQL

# Project Overview
This project analyzes a multi-table e-commerce dataset to uncover customer behavior, sales trends, payment patterns, and logistics performance across Brazil.  
The analysis was performed using **SQL** with a focus on **business insights and actionable recommendations**, simulating a real-world data analyst role at Target.

--

# Dataset Description
The dataset consists of **8 CSV files** representing different business entities:

- `customers.csv` – customer demographic and location data  
- `orders.csv` – order lifecycle and timestamps  
- `order_items.csv` – product price and freight details  
- `payments.csv` – payment type and installment information  
- `reviews.csv` – customer feedback and ratings  
- `products.csv` – product attributes  
- `sellers.csv` – seller location details  
- `geolocation.csv` – geographic mapping data  

--

# Objectives
- Perform exploratory data analysis (EDA)
- Analyze order growth and seasonality
- Study customer distribution across cities and states
- Evaluate revenue, freight cost, and economic impact
- Analyze delivery performance vs estimated timelines
- Understand payment behavior and installment trends

--

# Key Analyses Performed
- Time range of orders and geographic coverage
- Yearly and monthly order growth trends
- Order distribution by time of day
- State-wise revenue and freight analysis
- Delivery time and delay vs early delivery analysis
- Payment type trends (credit card, UPI, vouchers, etc.)
- Installment-based purchasing behavior

--

# Tools & Technologies
- **SQL (BigQuery)**
- Joins, aggregations, CTEs, date functions
- Window functions for advanced analysis

--

# Business Impact
- Identified logistics and optimization opportunities
- Provided data-driven recommendations for pricing, delivery, and payment strategies

--

# Conclusion
This case study demonstrates end-to-end SQL analytics skills, business thinking, and the ability to translate raw data into meaningful insights and recommendations.
